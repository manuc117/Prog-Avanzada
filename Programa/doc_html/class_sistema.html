Descripción detallada

Clase capaz de ejecutar el software de procesamiento de imágenes digitales e ineractuar con el usuario.

Tiene la capacidad de ejecutar todas las acciones que hacen funcionar el software. Posee un espacio de trabajo y una ventana de graficación.
