Descripción detallada

Clase que permite binarizar una Imagen.

Hereda el método de la clase ProcesadorDeImagen y lo redefine para binarizar una Imagen, siempre y cuando esta sea de intensidades. Es capaz de obtener el nivel a partir del cual se quiere binarizar la imagen.
