Descripción detallada

Clase capaz de aplicarle el algoritmo del pintor a una imagen.

Permite, a partir de un pixel inicial, buscar los vecinos de este teniendo en cuenta cierto nivel de tolerancia, especificado por el usuario. Si identifica un pixel como vecino, lo pinta con un color predeterminado (azul).
