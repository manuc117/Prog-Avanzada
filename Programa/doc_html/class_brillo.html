Descripción detallada

Clase que permite aumentar o reducir el brillo de una Imagen.

Hereda el método de la clase ProcesadorDeImagen y lo redefine para aumentar o reducir el brillo de una imagen, a partir de un valor indicado por el usuario o uno por defecto (50).

Autor
Manuela Calvo
Fecha
Fecha de última modificación: 15-02-2022
Precondición
Se debe especificar si el objeto construido va aumentar o reducir el brillo.
