Descripción detallada

Clase que forma un pixel de una imagen.

Representa el pixel de cualquier tipo de imagen (RGB, de intensidades o monocromática). Posee valores enteros de rojo, verde y azul, los cuales puede devolver. Además puede calcular la intensidad del pixel a partir de estos tres valores.
