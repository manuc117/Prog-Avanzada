Descripción detallada

Clase capaz de ejecutar el sistema de procesamiento de imágenes digitales.

Tiene la capacidad de ejecutar todas las acciones que hacen funcionar el software. Posee un espacio de trabajo, un graficador y un puntero gestor de archivos.
