Descripción detallada

Clase capaz de gestionar los diferentes tipos de archivos PNM.

Hereda de GestorDeArchivos los métodos leer y guardar archivos, y de almacenar la ruta del último archivo leído. Además, a partir del código de identificación de una imagen, puede obtener la extensión correspondiente.
